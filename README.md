## Installation

Dillinger requires [Node.js](https://nodejs.org/) v16+ to run.

Install the package.

```sh
git clone https://github.com/AkumaSama/BrainCellCounter.git
cd BrainCellCounter
cd api
npm i
cd ../bot
npm i
```

## Start production

To start each thing

```sh
in each directory
API :
node run start

BOT : 
node run start
```
