# BrainCellCounter

This repository contains
- expressjs API
- Discord v13.3.1 Bot
- ✨Magic ✨

## Installation

Dillinger requires [Node.js](https://nodejs.org/) v16+ to run.

Install the package.

```sh
git clone https://github.com/AkumaSama/BrainCellCounter
cd BrainCellCounter
cd api
npm i
cd bot
npm i
```

To start each thing

```sh
in each directory
API :
node run start

BOT : 
node run start
```

Yep it's the same command :)
